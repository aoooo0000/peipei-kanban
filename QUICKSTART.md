## 🎮 四大功能頁面

### 📋 看板（Kanban）
- **用途：** Notion 任務管理
- **功能：** 
  - 拖曳卡片移動狀態
  - 新增/編輯/刪除任務
  - 依狀態分欄顯示
- **數據同步：** 15 秒自動刷新

### 📊 總覽（Dashboard）
- **用途：** Agent 狀態監控
- **功能：**
  - 🟢 Agent 狀態燈（idle/thinking/acting）
  - 📈 任務統計（總數 + 各狀態數量）
- **數據刷新：** 3 秒（狀態）/ 10 秒（任務）

### 📄 文件（Docs）
- **用途：** Workspace Markdown 文件瀏覽
- **功能：**
  - 文件列表（System/Docs 分類）
  - 點擊檢視完整內容
  - Markdown 渲染（表格、代碼、引用）
- **掃描範圍：** `~/clawd/*.md` + `~/clawd/memory/*.md`

### 📝 日誌（Log）
- **用途：** Agent 活動記錄
- **功能：**
  - 時間軸顯示
  - 類型篩選（Agent/任務/文件/系統）
  - 展開查看詳細資訊
- **數據刷新：** 5 秒

## 🔧 常見問題

### Q1: Vercel 上文件/日誌顯示空白？
**A:** Vercel 無法存取 Mac mini 的檔案系統，這是正常的。使用 Tailscale 本地連線可解決。

### Q2: 如何安裝 PWA 到手機主畫面？
**A:** 
1. 用 Chrome/Safari 開啟網站
2. 點選「新增到主畫面」
3. 圖標會出現在主畫面，像原生 App 一樣啟動

### Q3: Agent 狀態一直顯示 idle？
**A:** 
- Vercel 版本：正常，因為無法讀取 `~/.openclaw/cron/jobs.json`
- 本地版本：表示目前沒有 cron job 在執行

### Q4: 任務拖曳在手機上不順？
**A:** 長按卡片上方的「⠿」圖標 0.25 秒再拖曳，避免誤觸。

### Q5: 文件內容字太小？
**A:** 雙指縮放或在設定中調整瀏覽器字型大小。
